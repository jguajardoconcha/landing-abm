## 📋 Contenido

- `index.html` - Página principal (completa y funcional)
- `images/` - Carpeta con todas las imágenes del diseño

## 🚀 Cómo subir a GitHub Pages (Paso a paso)

### 1. En GitHub.com:
Usa la cuenta que ya creaste (jguajardoconcha)

### 2. Sube los archivos:
- En tu repositorio `landing-abm`, haz clic en **"Add file"** → **"Upload files"**
- Selecciona la carpeta `landing-abm` completa (arrastrar y soltar)
- Haz clic en **"Commit changes"**

### 3. Activa GitHub Pages:
- Ve a **Settings** del repositorio
- En el menú izquierdo, busca **"Pages"**
- En "Build and deployment", selecciona **"Deploy from a branch"**
- Elige la rama **"main"**
- Click en guardar

### 4. Tu URL pública será:
```
https://jguajardoconcha.github.io/landing-abm
```
